Linkbucks Buster is a small tool to block websites using linkbucks. It will analytic the website , take you to the target website.


Linkbucks Buster have added some websites to the blacklist.To add more:

  • * open the website using linkbucks
  • * right click, open context menu, choose "Bust This Linkbucks Site"
  • * refresh, watch the effect

you can edit the black list in the options page. You will see it in chrome extensions page.
